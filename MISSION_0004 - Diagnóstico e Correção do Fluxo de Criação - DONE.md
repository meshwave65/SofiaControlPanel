# Missão 0004: Diagnóstico e Correção do Fluxo de Criação

## Descrição
Esta missão foca em identificar o motivo pelo qual a criação de novas missões via UI está falhando ou não persistindo como esperado, garantindo a integridade dos dados no banco de dados da API SOFIA.

## Objetivos
1. Diagnosticar falhas no processo de `POST` de novas tarefas.
2. Validar a persistência de títulos e status iniciais.
3. Corrigir qualquer dessincronia entre a UI e a API.
